=== AREA PRIVADA DE MODIFICACION DE PUNTOS ===

Todas las modificaciones de la guerra de bandas se hacen en:

    /data/data.js

Aquí puedes:
- Agregar nuevos eventos
- Modificar puntos de cualquier banda
- Agregar actividades diarias
- Actualizar posiciones
- Cambiar nombres de bandas

Una vez modificado:
1. Guarda el archivo data.js
2. Sube los cambios a GitHub
3. La web pública se actualiza automáticamente




=== SUBIRLO A LA WEB ===

git add .
git commit -m "aquí pones que cambios agregaste"
git push
